# Platform required
  This app is based on react native cli, so 
  * react native cli,
  * Android Studio,
  * X-Code,
  * Python 2.7 or above ,
  * Jdk, Node Js and Cocapods should
  be installed and configure in your sytem.
  
 # How to Run
  For run android use command # npm run android
  For iOS use command npm # run ios
